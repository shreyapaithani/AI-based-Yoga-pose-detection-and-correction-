# AI-Based Real-Time Yoga Pose Detection and Correction

## Overview
This project uses Computer Vision and Machine Learning techniques to detect yoga poses in real time and provide posture correction feedback.

## Features
- Real-time pose detection
- Body landmark tracking
- Pose classification
- Posture correction suggestions
- Live webcam integration

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
- Machine Learning

## How It Works
1. Capture video from webcam.
2. Detect body landmarks using MediaPipe.
3. Compare detected pose with predefined yoga poses.
4. Generate posture correction feedback.
5. Display results in real time.

## Applications
- Yoga training
- Fitness monitoring
- Home workout assistance

## Future Improvements
- More yoga poses
